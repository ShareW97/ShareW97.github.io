Hexo博客搭建

发表于 2020-07-06 | 分类于 Git
字数统计: | 阅读时长 ≈

心心念念了很久要搭建一属于自己的blog，但是自己懒癌晚期- -拖了蛮久…现在之所以想起也是因为这几天突然一下子对Github心血来潮，Lizzie关于Mac的一些问题着实让我这个计算机萌新折腾了半天…话不多说，接下来简述下心得体会- -

环 境

Win10 x64

萌新准备工作(勿喷- -)

Github(各种开源)、CSDN(好像喷的有点多)、SSR(快乐源泉)

开始操作

1. 安装Git、Nodejs

  • 1. Git(点击进入官网) 附CSDN
  • 2. Nodejs(点击进入官网) 如果不想占用c盘空间建议在别的空的盘安装，并且在安装node.js的位置创建global和cache两个文件夹。具体附CSDN

2. 安装Hexo

  • 1.附上我最初看的CSDN，然而小白我并没有5分钟完成所有…这里面提到的brew命令也是我的一个辛酸史啊….
  • 2.Hexo安装用的是$ npm install hexo-cli -g命令，安装确实是成功了，但是只会输入$ hexo -v显示command not found..这个问题耽误了很长时间…最终的解决方案是添加个环境变量(用户变量PATH + 编辑 + 新建 + 输入hexo路径)

3. 编写发布

  • 1.git clone大法， 不得不说Hexo上有太多好看的theme了

  • 2.Github+SSH， 之前没有弄SSH，所以一直不能用username.github.io登陆…只能本地localhost:4000(数字又不吉利)

小结

第一次写这方面的算是技术帖吧…小萌新的第一次尝试，太晚了，实在熬不动了，有漏的地方还望见谅。
